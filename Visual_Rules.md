# Zhemin.ltd 网页视觉设计规范

| 类别 | 层级／对象 | 视觉规范 |
|---|---|---|
| 分析范围 | 页面 | 基于 `zhemin.ltd` 当前线上 CSS；首页及代表性内部文章页 `/articles/copper-supply-demand` |
| 字体 | 英文 | 首选 `Inter`；未安装时使用系统无衬线字体：macOS/iOS 通常为 SF Pro，Windows 通常为 Segoe UI |
| 字体 | 中文 | 系统无衬线字体；显式回退顺序为 `PingFang SC` → `Hiragino Sans GB` → `Microsoft YaHei` → `sans-serif`。因此 macOS 通常显示苹方，Windows 通常显示微软雅黑 |
| 字体 | 完整字体栈 | `Inter, ui-sans-serif, -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`。网站未加载自托管 Web Font，实际字体会受操作系统影响 |

| 字体样式：首页 | 个人介绍 | 16px；常规字重；行高 26px；颜色：浅色 `#4F4F4F`、深色 `#C7C7C7` |
| 字体样式：首页 | 栏目标题 | 14px；字重 500；行高 20px；颜色：浅色 `#8A8A8A`、深色 `#8D8D8D` |
| 字体样式：首页 | 项目／文章名称 | 16px；常规字重；约 26px 行高；颜色：浅色 `#1A1A1A`、深色 `#F2F2F2` |
| 字体样式：首页 | 描述、年份 | 14px；常规字重；约 23px 行高；颜色：浅色 `#8A8A8A`、深色 `#8D8D8D` |

| 字体样式：首页 | 语言按钮 | 13px；字重 500；字间距 `0.04em` |
| 字体样式：文章页 | 文章标题 | 22px；字重 600；行高 28px；颜色：浅色 `#1A1A1A`、深色 `#F2F2F2` |
| 字体样式：文章页 | 返回链接、日期 | 14px；行高 22px；颜色：浅色 `#8A8A8A`、深色 `#8D8D8D` |
| 字体样式：文章页 | 正文、列表 | 16px；字重 400；行高 24px；颜色：浅色 `#1A1A1A`、深色 `#F2F2F2` |

| 字体样式：文章页 | 四级标题 H4 | 16px；字重 550；行高 24px；上间距 24px、下间距 8px |
| 字体样式：文章页 | 图片说明、表格 | 14px；行高 22px；图片说明使用弱化色；表头字重 600 |
| 字体样式：文章页 | 目录 | 13px；行高 20px；目录名称字重 500；二、三级目录分别缩进 14px、28px |
| 字体样式：文章页 | 代码 | 行内代码约 14.4px；代码块 14px／22px；字体为 `ui-monospace, SFMono-Regular, Consolas, monospace` |
| 布局 | 桌面端页面容器 | 最大总宽度 760px，水平居中；左右内边距各 22px，因此最大可阅读内容宽度为 **716px** |
| 布局 | 桌面端页边距 | 视口宽度超过 760px 后，外侧页边距为 `(视口宽度 − 760px) ÷ 2`；内容到容器边缘另保留 22px |
| 布局 | 桌面端垂直留白 | 页面顶部 112px；底部 56px |
| 布局 | 移动端 | 断点为 800px；左右外边距各 20px，可阅读宽度为 `视口宽度 − 40px`；顶部 80px、底部 40px |
| 布局 | 首页首屏 | 桌面端为文字＋肖像双栏网格：文字约占 1.9fr、肖像约占 0.55fr；栏间距在 40–88px 之间响应变化 |
| 布局 | 肖像 | 桌面端最大宽度 210px，比例 4:5；≤800px 为 112px；≤520px 为 96px；圆角 8px |
| 布局 | 首页列表 | 列表向内容区左右各延伸 12px；每一行最小高度 52px，内边距 13px × 12px，标题与年份间距 24px |
| 布局 | 文章头部 | 标题区与正文间距 48px；移动端缩小至 40px；返回首页链接下方间距 32px |
| 布局 | 文章正文 | 相邻正文段落间距 16px；列表上、下间距 16px，左缩进 24px；列表项间距 6px |
| 布局 | 文章目录 | 宽屏下固定在左侧，宽 240px、视觉高度 294px、垂直居中；1221–1320px 缩为 200px；≤1220px 进入正文流并置于文章上方 |
| 布局 | 页脚 | 首页与末段内容间留白 160px，移动端 112px；文章页正文与页脚间距 112px，移动端 88px |
| 色彩 | 浅色主题 | 背景 `#FCFCFC`；主文字 `#1A1A1A`；正文辅助文字 `#4F4F4F`；弱化文字 `#8A8A8A`；次级弱化 `#B8B8B8`；边框 `#E3E3E3`；悬停背景 `#EEEEEE` |
| 色彩 | 深色主题 | 背景 `#171717`；主文字 `#F2F2F2`；正文辅助文字 `#C7C7C7`；弱化文字 `#8D8D8D`；次级弱化 `#686868`；边框 `#343434`；悬停背景 `#292929` |
| 组件 | 圆角 | 肖像、按钮、列表悬停背景、引用块右侧圆角主要统一为 8px；行内代码为 4px；整体偏克制 |
| 组件 | 引用块 | 上间距 24px；左侧 2px 边线；内边距 16px × 18px；背景使用悬停底色；字号 16px／24px |
| 组件 | 表格 | 宽度 100%；上下间距 24px；单元格下方 1px 分隔线；内边距 `8px 12px 8px 0`；长内容允许换行 |
| 组件 | 图片 | 最大宽度 100%，高度自适应、水平居中、不设圆角；图片块上下间距 24px；图注与图片间距 8px |
| 组件 | 链接 | 正文链接使用主文字色；下划线默认采用边框色，悬停后加深；下划线偏移 4px |
| 组件 | 控件 | 语言与主题按钮高 38px、最小宽 38px、圆角 8px；固定在页面右上方，桌面端与内容右边缘对齐 |
| 交互 | 悬停 | 首页条目悬停时出现浅灰背景，右上箭头由左下方向原位移动并淡入；持续时间约 150ms |
| 交互 | 焦点 | 键盘焦点使用 2px 实线轮廓，外偏移 2px；颜色随明暗主题自动切换 |
| 动效 | 页面进入 | 内容由下方 10px 上移并淡入；时长 500ms；不同区块依次延迟约 40–300ms |
| 动效 | 无障碍 | 检测到“减少动态效果”偏好时，关闭平滑滚动，并把动画、过渡压缩至约 0.01ms |
| 视觉原则 | 整体风格 | 极简、低对比、无衬线、窄内容栏；主要依靠字号、字重、留白和灰阶建立层级，装饰元素极少 |
